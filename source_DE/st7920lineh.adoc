===== ST7920lineh

*Syntax:*
----
    ST7920lineh ( Xpos, Ypos, xUnitsStyle, )
----
*Explanation:*

This command draws a horizontal line with the specific style. The style
can be ON or OFF. Default is ON.

This is called by the GLCD common routines.
