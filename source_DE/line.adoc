==== Line

*Syntax:*
----
    Line(LineX1,LineY1, LineX2, LineY2, Optional LineColour = 1)
----
*Explanation:*

Draws a line on a GLCD from pixel X1, Y1 location to pixel X2,Y2 location.

*Example:*
----
    #include <glcd.h>

    line 0,0,127,63
    line 0,63,127,0
    line 40,0,87,63
    line 40,63,87,0
----
