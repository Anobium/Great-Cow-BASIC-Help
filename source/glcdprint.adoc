==== GLCDPrint

*Syntax:*
----
    GLCDPrint(PrintLocX, PrintLocY, Print_Byte )    ',or
    GLCDPrint(PrintLocX, PrintLocY, Print_Word )    ',or
    GLCDPrint(PrintLocX, PrintLocY, Print_Long )    ',or
    
    GLCDPrint(PrintLocX, PrintLocY, Print_String )
----
